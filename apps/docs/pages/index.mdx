## Builder Utils

Integrate nouns builder functionality into your React website.

### FAQ

##### Who created this project?

This project was created by a working group from [Public Assembly](https://twitter.com/pblcasmbly).

##### How can I contribute to this project?

If you're someone with an idea on how to improve this project, please get involved. Anyone with a GitHub account can [create an issue](https://github.com/public-assembly/flexible/issues/new) detailing what they'd like to see from this app. Additionally, there are a number of [existing issues](https://github.com/public-assembly/flexible/issues) that are open to contributions.
