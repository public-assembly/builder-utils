## useTokenMetadata

Hook for accessing the name and thumbnail of a token given its id.

```jsx
import { useTokenMetadata } from '@public-assembly/builder-utils'
```

## Usage

```jsx
import { useTokenMetadata } from '@public-assembly/builder-utils'

function App() {
  const { tokenName, tokenThumbnail } = useTokenMetadata('1' // tokenId)

  return <>
      <p>{tokenName}</p>
      <img src={tokenThumbnail}></img>
    </>
}
```

## Returns

```jsx
tokenName?: string
tokenThumbnail?: string
```
